# Contributing to OfflineMessenger

Contributions are welcome! If you find a bug or have a suggestion, please open an issue in this repository.

## Localization
We are actively looking for help with localization. Contributions for the following languages are welcome:
- Czech (CZ)
- Hungarian (HU)
- Polish (PL)
- German (DE)

## Pull Requests
Before submitting a major change or feature, please contact Ing. Jozef Antol, MBA to discuss your proposal.
