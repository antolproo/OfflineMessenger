# OfflineMessenger — Roadmap

## Current version (v0.1 — Slovak adaptation)
- Slovak (sk) UI localization
- Adapted README for Central European crisis communication context
- Increased default TTL from 7 to 20 hops (for urban Slovak deployment)

## Planned features
- Czech (cs) localization
- Crisis-specific channel presets (fire, flood, medical emergency, evacuation)
- Integration guidance for Meshcore LoRa bridge nodes
- HAM radio gateway documentation
- Offline map tile support (future)
- Voice message support (future development, pending follow-on funding)
- Photo transmission (future development)

## Target deployment context
- Rural municipalities, Košice region, Slovakia
- 3-layer resilience architecture:
  1. OfflineMessenger (Bluetooth mesh) — mass user layer
  2. Meshcore LoRa nodes — medium-range bridge layer  
  3. HAM radio via ARES SZR — long-range regional layer

## Research integration
- Part of rural resilience research ecosystem
- Linked to Domáca Pevnosť citizen preparedness guide
